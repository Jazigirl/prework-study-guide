HTML

  • HTML is used to create structure of a webpage.
  • The head element contains information about the webpage.
  • The head elements include: DOCTYPE html, html, head, meta, and title.
  • The head element contains information and metadata that helps the browser to render the page correctly.
  • The body element represents the visible content shown to the user.
  • Body element is an example of a semantic HTML element.
  • Child elements are elements that are nested inside other elements.
  • Elements are composed of an opening tag and a closing tag with angle brackets.

CSS

  • Cascading Style Sheets is a programming language used by developers to define how webpage content is presented to users.
  • There are 3 ways to style a webpage using CSS: Inline CSS, Internal Style Sheet, and External Style sheet.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Can combine 2 selectors into single CSS rule with a for declarations to affect both selectors.

Git

  • The "git" command must precede every statement.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • Add update to current working branch using "git add -A".
  • When files are staged to commit enter :"git commit" with commit message that describes what the commit contains.

JavaScript

  • JavaScript is a high-level programming language that allows users to interact with websites.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Comments are a snippet of text that can be added along with code.
  • An array is a group of related data in a single variable.
  • An index is a location identified with a unique number.
  • A loop uses the predictable pattern of indices to perform a task on all items in an array.
  • Commenting out code makes it inactive without deleting it.
  • A function is a set of instructions that tells the computer how to perform a certain task.
